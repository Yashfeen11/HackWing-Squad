# Water Leakage & Theft Detection - Round 2

## Overview
Production-ready water monitoring system for detecting leakage and theft using flow meter data.

## Architecture Components

### 1. Data Simulation
Simulates sensor data from ESP modules and flow meters for testing.

### 2. Processing Engine
Core detection engine for baseline learning, anomaly detection, and classification.

### 3. Classification & Alerts
Classifies anomalies as leakage, theft, or normal and triggers alerts.

### 4. Storage & Dashboard
Data storage and visualization dashboard for monitoring.
